# spec-manager

This skill manages and displays the application's specification documents. When specifications become clearer during the early development phase or when major specification changes occur, it updates the documents and notifies users.

## Installation

```
/plugin marketplace add hiroshi75/ccplugins
/plugin install spec-manager-plugin@hiroshi75
```

## Usage

```
/spec-manager-plugin:update-spec
```

Or if you prefer other languages:

```
/spec-manager-plugin:update-spec 日本語で
```
